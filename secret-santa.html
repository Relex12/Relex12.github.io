Secret Santa JS

Qu'est-ce que c'est ?

Secret Santa JS est application web monopage qui vous permet de réaliser un Père Noël Surprise.

Chaque participant glisse son nom dans une boîte puis chacun pioche le nom de la personne à laquelle il devra offrir un cadeau.

Avec la version papier :

      Si une personne pioche son propre nom, il faut recommencer tout le tirage,
      Si plusieurs personnes ne se connaissent pas bien, trouver un cadeau peut être difficile,
      Si une personne pioche le même participant chaque année, cela peut être redondant.

Pour éviter tout ça, utilisez Secret Santa JS.

Comment l'utiliser ?
  1. Ajoutez les participants un à un ci-dessous,
  2. Si besoin, modifiez la liste des participants avec ✏️ et 🗑️,
  3. Pour chaque participant, cochez les personnes qu'il ne pourra pas recevoir,
  4. Lancez le tirage,
  5. Chaque participant révèle sa carte à tour de rôle.
